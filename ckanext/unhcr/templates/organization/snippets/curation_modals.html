{# Deposit #}

Deposit Dataset

You can use the Data Deposit to submit data that is not yet ready for publication. While in the Data Deposit, datasets will be only visible to yourself, the curation team and the site admins.

After depositing the dataset, you will be able to make additional changes. Once you are ready, you can submit it to the curation team, which will work on the dataset metadata to make it meet the RIDL requirements. A designated curator will be assigned to the dataset as contact point. At this point you won't able to make further changes to the dataset, but the curator will be able to release it back in order for you to do so.

If the dataset is approved and passes the validation it will be published on the site. Please select below the suggested destination Data Container and the Access Level for the dataset you are submitting.
